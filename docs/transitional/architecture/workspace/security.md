# Workspace Security

Canonical security and scope model for workspace execution.

## Runtime Surfaces
- Governance contracts: `include/yai/policy/governance/`
- Runtime policy/grants: `lib/runtime/policy/`, `lib/runtime/grants/`
- Enforcement modules: `lib/runtime/enforcement/`, `lib/governance/`

## Covers
- privilege scope boundaries
- session and shell binding constraints
- policy application boundaries
- governance/evidence alignment
