# D18.2 Source-of-Truth Contract Map

## Contract Model
- Every `primary_for` topic has exactly one live owner.
- Non-owner docs on the same theme must be marked `support`, `procedural`, or `reference` and declare `depends_on`.
- Historical material is excluded from owner election.
