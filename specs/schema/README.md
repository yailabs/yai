# schema/

`schema/` contains canonical structural schemas validated by governance tooling.

## Ontology posture

These schemas define record/object shapes.
Operational bindings belong to `../contracts/`.

## Stability posture

Schema file names and compatibility lines are retained where valid.
Semantic descriptions and usage guidance are updated when ontology shifts.
