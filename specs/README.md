# specs

Canonical specification surface.

- `authority/`: normative authority corpus
- `taxonomy/`: controlled vocabularies
- `ontology/`: conceptual relationships and structures
- `registries/`: operational runtime/tooling registries
- `protocol/`: exchange contract payloads
- `schemas/`: canonical validation schemas
- `ingestion/`: source acquisition and normalization pipeline
- `compliance/`: normative catalog packs
- `overlays/`: contextual overlay profiles and composition indexes
- `policies/`: global policy composition grammar
- `manifests/`: runtime/tooling publication surfaces
- `vectors/`: validation/audit vectors
