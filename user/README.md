# user

Canonical operator/user execution environment.

- `bin/`: user-facing entrypoints (`yai`, `yai-ctl`, `yai-sh`)
- `shell/`: interactive shell runtime (parse/render/help/watch/term/core)
- `commands/`: operator command families (runtime/container/policy/graph/network/data/observability/orchestration)
- `include/yai/`: user/shell/commands public headers
- `share/`: completions, prompts, themes and man assets
