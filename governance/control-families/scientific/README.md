# Scientific

`scientific` is a control family focused on reproducibility, experiment integrity,
parameter governance, and publication controls for scientific outputs.

## Pillar role

Scientific is a semantic control pillar for wave-1 verticalization:
- parameter and reproducibility controls
- high-risk review and oversight composition (AI Act)
- evidence obligations for experiment rationale and provenance

## Primary specialization set

- `experiment-configuration`
- `parameter-governance`
- `reproducibility-control`
- `black-box-evaluation`
- `dataset-integrity`
- `result-publication-control`

Legacy seed payload remains in `transitional/domain-family-seed/` for bridge compatibility.
