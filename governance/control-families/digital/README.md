# Digital

`digital` is a control family focused on external effects across network, publication, retrieval,
and digital sink control.

## Pillar role

Digital is a semantic control pillar for wave-1 verticalization:
- constrains egress/publication behavior
- composes strongly with GDPR and supply-chain overlays
- enforces evidence traceability for external effects

## Primary specialization set

- `network-egress`
- `remote-publication`
- `external-commentary`
- `artifact-distribution`
- `remote-retrieval`
- `digital-sink-control`

Legacy seed payload remains in `transitional/domain-family-seed/` for bridge compatibility.
