# packs

Versioned or materialized governance bundles.

`packs/` is not the primary authoring surface for base policy semantics.
It stores packaged artifacts that are distributed or referenced as bounded
bundle outputs.

Current canonical pack focus:

- `packs/compliance/*` for versioned compliance bundle materializations

Primary policy semantics remain in:

- `compliance/`
- `overlays/`
- `classification/`
- `families/`
- `specializations/`
