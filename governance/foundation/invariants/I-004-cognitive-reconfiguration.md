# I-004 — Cognitive Reconfiguration

Cognitive reconfiguration is integrated and governed.

## Invariant

When cognitive validity fails, `brain` MUST enter explicit reconfiguration flow under governance. Execution continuation without such flow is invalid.

## Required properties

- invalidation signal is explicit and evidentiary
- suspension or constrained mode is decision-bound
- old/new cognitive configuration references are recorded
- resumption requires renewed authority evaluation
- reconfiguration cannot self-authorize effects

## Ontology alignment

`brain` is an internal runtime plane, not an external subsystem.
Historical "brain" framing is deprecated as a primary normative model.
