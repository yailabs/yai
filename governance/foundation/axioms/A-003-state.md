# A-003 — State Discipline

State is derived from governed transitions.
State is not self-authorizing.

YAI state model has three planes:
- sovereign state (`core`): authority, lifecycle, workspace sovereignty
- execution state (`exec`): bounded operational/effect state
- cognitive state (`brain`): cognitive configuration and memory state under governance

## Normative statements

- State changes MUST be traceable to authorized transitions.
- Workspace sovereignty resides in `core` and governs admissible transitions.
- `exec` state may optimize execution but cannot redefine sovereign truth.
- `brain` state may adapt cognition but cannot bypass authority.
- Hidden state that cannot be audited is invalid as normative state.

## Lifecycle coupling

Initialization, run, suspend, resume, and shutdown must preserve state traceability and boundary integrity across all planes.
