# L0 — Vault Boundary

## Boundary role

L0 is the foundational state surface boundary.
It defines canonical shared state layout and stable low-level identifiers required for cross-plane coherence.

## Primary semantics

- canonical memory/record layout constraints
- stable ID and field interpretation contract
- compatibility and integrity rules

## Ontology alignment

L0 is below plane semantics.
It is consumed by `core`, `exec`, and `brain` but does not define their authority model.

## Non-goals

L0 does not define policy, cognition, planning, or authority decisions.
