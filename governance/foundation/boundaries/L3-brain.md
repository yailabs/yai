# L3 — Cognitive Boundary (historical alias: mind)

## Boundary role

L3 is the `brain` boundary.
It defines cognitive proposal, planning, memory usage, and reconfiguration under governance.

## Primary semantics

- proposal and planning surfaces
- cognitive state and reconfiguration flow
- memory/retrieval semantics under governance
- cognitive evidence and attribution

## Ontology alignment

Primary term: `brain`.
Historical label `mind` is migration alias only.

## Prohibited patterns

- direct execution authority in cognitive surfaces
- silent reconfiguration
- external effects initiated without sovereign decision path
