# Governance Implementation Domain (Canonical)

`lib/governance/` is the canonical implementation domain for governance logic
in the unified YAI repository.

Canonical implementation slices include:

- `classification/`
- `discovery/`
- `loader/`
- `mapping/`
- `resolution/`
- `policy_effects*`
- `policy/compliance/`
- `policy/overlays/`
- `authority_model/`
- `internal/`

`lib/governance/` is the only canonical implementation namespace for policy-engine logic.
