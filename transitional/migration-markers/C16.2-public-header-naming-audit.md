# C16.2 Public Header Naming Audit

Date: 2026-03-11
Owner: yai
Status: applied

## Scope

Public API naming pass across `include/yai/**` with header-by-header cutover for legacy
hybrid names from edge/orchestration/providers surfaces.

## Renamed (implemented)

### Edge

- `include/yai/edge/action_point.h` -> `include/yai/edge/actions.h`
- `include/yai/edge/edge_binding.h` -> `include/yai/edge/binding.h`
- `include/yai/edge/edge_services.h` -> `include/yai/edge/services.h`
- `include/yai/edge/edge_state.h` -> `include/yai/edge/state.h`
- `include/yai/edge/local_runtime.h` -> `include/yai/edge/local.h`
- `include/yai/edge/source_ids.h` -> `include/yai/edge/ids.h`
- `include/yai/edge/source_plane_model.h` -> `include/yai/edge/source_plane.h`

### Orchestration

- `include/yai/orchestration/agent_contract.h` -> `include/yai/orchestration/agent_binding.h`
- `include/yai/orchestration/engine_bridge.h` -> `include/yai/orchestration/runtime_bridge.h`
- `include/yai/orchestration/engine_cortex.h` -> `include/yai/orchestration/runtime_model.h`
- `include/yai/orchestration/network_gate.h` -> `include/yai/orchestration/network_policy.h`
- `include/yai/orchestration/provider_gate.h` -> `include/yai/orchestration/provider_policy.h`
- `include/yai/orchestration/resource_gate.h` -> `include/yai/orchestration/resource_policy.h`
- `include/yai/orchestration/storage_gate.h` -> `include/yai/orchestration/storage_policy.h`
- `include/yai/orchestration/source_ingest.h` -> `include/yai/orchestration/ingestion.h`

### Providers

- `include/yai/providers/providers.h` -> `include/yai/providers/catalog.h`

## Consumer cutover

All direct include consumers under `include/`, `lib/`, `tests/`, `cmd/`, and `tools/`
were updated to the renamed headers.

## Guardrail added

- `tools/validate/validate_public_header_naming.py`
  - fails if legacy headers are reintroduced
  - fails if legacy include tokens are reintroduced in code/test/tooling surfaces

## Deferred to C16.3+

- Internal type/function symbol renaming still using `exec`, `edge_state`, `edge_binding`,
  and related historical internal identifiers.
- Further umbrella-header minimization decisions (`graph.h`, `orchestration.h`, `runtime.h`).
- Protocol de-dup pass on compatibility wrappers (`include/yai/protocol/*.h` wrappers).
